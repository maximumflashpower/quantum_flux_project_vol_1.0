Quantum Flux Backup Dashboard v7 Corporate

Panel de referencia para usar la consola de backups Quantum Flux desde PowerShell.

1. Abrir la consola principal

Ejecutar desde PowerShell (en la carpeta raiz del proyecto):

cd C:\Users\joseb\Downloads\chatgpt_organized_quantum_flux_project\quantum_flux_project_vol_1.0 .\quantum_flux_backup_system\qf-menu.ps1

2. Tareas principales

Opciones disponibles en la consola:

  • 1 Crear Backup Completo
  • 2 Backup + Release en GitHub
  • 3 Backup Incremental
  • 6 Restaurar Backup
  • 7 Ver Log del Sistema
  • 8 Ver Estado del Sistema
  • 9 Abrir carpeta de Restauraciones
  • 10 Limpiar Backups Antiguos

3. Carpetas importantes

Proyecto:

C:\Users\joseb\Downloads\chatgpt_organized_quantum_flux_project\quantum_flux_project_vol_1.0

Backups (.zip) dentro del proyecto: quantum_flux_backup_YYYY-MM-DD_HH-mm-ss.zip

Restauraciones:

C:\Users\joseb\Documents\QuantumFluxRestore

4. Limpiar backups antiguos

La opcion 10 de la consola ejecuta una limpieza automatica:

  • Mantiene los ultimos 6 archivos quantum_flux_backup_*.zip.
  • Mantiene las ultimas 6 carpetas de restauracion en QuantumFluxRestore.

Puedes modificar el numero de items a mantener editando la variable $keepCount dentro de Clean-OldBackups en qf-menu.ps1.

5. Estado del sistema

La opcion 8 muestra:

  • Version Quantum Flux.
  • Ruta del proyecto.
  • Ultimo backup detectado.
  • Cantidad de registros en el log de backups.
  • Si git y gh estan instalados.
  • Estado general del sistema.
